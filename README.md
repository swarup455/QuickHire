# Quick Hire

Quick Hire is a modern full-stack job portal designed to connect talented job seekers with employers through a fast, intuitive, and responsive platform. It provides a seamless experience for discovering opportunities, posting jobs, and managing the hiring process.

The project is currently under active development. New features, backend integrations, and improvements are continuously being added.

---

## Features

### Job Seekers
- Browse the latest job opportunities
- Search and filter jobs
- View detailed job descriptions
- Apply for jobs
- Manage personal profile
- Track job applications

### Employers
- Create and manage company profiles
- Post and manage job listings
- View and manage applicants
- Access a dedicated employer dashboard

---

## Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- React Router DOM
- shadcn/ui
- Lucide React
- Embla Carousel

### Backend (In Development)
- Node.js

- MongoDB

- JWT Authentication

## Installation

### Clone the repository

```bash
git clone https://github.com/your-username/quick-hire.git
```

### Navigate to the project

```bash
cd quick-hire
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
